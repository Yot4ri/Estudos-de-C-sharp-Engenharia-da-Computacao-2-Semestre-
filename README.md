# 📚 Estudos de C# – 2º Semestre da Graduação

Repositório destinado ao armazenamento dos exercícios, exemplos e atividades desenvolvidos durante a disciplina de Algoritmos no 2º semestre do curso de Engenharia da Computação.

Além de servir como registro da evolução acadêmica, este repositório também funciona como material de consulta para revisões e estudos futuros.

> **Execução:** Os exercícios foram desenvolvidos para execução em ambiente de terminal, utilizando o terminal integrado do Visual Studio.

> **Observação:** este repositório possui finalidade exclusivamente acadêmica e reúne atividades propostas ao longo da disciplina.

---

## 📌 Objetivo

Organizar todos os exercícios desenvolvidos durante o semestre em um único local, permitindo acompanhar a evolução dos conceitos estudados e a crescente complexidade dos problemas resolvidos.

Os principais objetivos deste repositório são:

- Registrar a evolução do aprendizado em C#;
- Centralizar os exercícios desenvolvidos durante a disciplina;
- Servir como material de consulta para estudos futuros;
- Praticar lógica de programação e resolução de problemas.

---

## 📂 Estrutura do repositório

O conteúdo está organizado em diretórios conforme os assuntos abordados durante o semestre.

Atualmente o repositório contém atividades relacionadas a:

- Diagnóstico de conhecimentos;
- Estruturas condicionais (`switch-case`);
- Estruturas de repetição;
- Métodos e funções;
- Recursividade;
- Coleções;
- Tratamento de exceções;
- Manipulação de arquivos e diretórios;
- Algoritmos de ordenação;
- Exercícios das avaliações (N2).

---
